Here you can read an explanation of the functions implemented in Circuit library.

The main class object which should be created at the beginning is a Circuit class:

Qubit = QCircuit.Circuit()

This class contains the following methods:

circuit.Circuit([tolerance]) The class containing references to nodes, elements, variables, variable-to-node mappings.

The element classes:

circuit.Capacitance(name[, capacitance]) Circuit element representing a capacitor.
