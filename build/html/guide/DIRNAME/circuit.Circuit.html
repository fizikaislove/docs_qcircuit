circuit.Circuit¶

class circuit.Circuit(tolerance=1e-18)¶

The containing references to nodes, elements, variables, variable-to-node mappings, in addition it contains all methods used for calculations.

__init__(tolerance=1e-18)¶

Default constructor. :param tolerance: capacitances below this value are considered to be computational errors when determining the inverse capacitance matrix. :type tolerance: float
